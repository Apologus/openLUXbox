# openLUXbox
DIY-Coop-Open-Source-**Spielekonsole** - Bauanleitung, 3D-Dateien, Dokumentation

*openLUXbox* ist eine Spielekonsole, die von dem *WLED Project* und dem *Open LED Race Project* inspiriert ist. Grundgedanke ist, eine einfache DIY-Hardware als Basis für **kooperative Spiele** zu nutzen. Die Steuerung erfolgt über 4 integrierte Push-Buttons.

Die Hardware besteht aus einer LED-Matrix (WS2812b, 16x16 LEDs), einem ESP8266, 4 Buttons und wenigen Kleinteilen. Das Gehäuse ist 3D-gedruckt. Bei renommierten Anbietern gekauft, liegen die Gesamtkosten bei maximal 40€.

Sämtliche Dokumente (Bauanleitung, Dokumentation, 3D-Druckdateien, Anleitung zum Programmieren) stehen frei zur Verfügung

Zum jetzigen Zeitpunkt gibt es 4 Spiele, die über einen Webflasher direkt aufgespielt werden können. Das Besondere ist, dass diese Spiele per Vibe Coding entstanden sind. Das heißt, die Beschreibung der gewünschten Spielefunktion wurde durch KI in Arduino-Code umgesetzt.
